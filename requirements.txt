# AI投資アシスタント 依存パッケージ
# Python 3.11 を前提（GitHub Actions ubuntu-latest + actions/setup-python@v6 で指定）
#
# Layer1（データ取得層）: requests, pandas, PyYAML
# Layer2（分析層）: pandas（テクニカル指標計算）, jsonschema
# Layer3（ニュース処理層）: google-genai（GeminiStructurer、コスト最適化のため採用）。
#   anthropicはClaudeStructurer用に維持（provider: claudeへ戻す場合の代替経路として残す）
# Layer4（永続化層／Google Drive連携）: google-api-python-client, google-auth, google-auth-httplib2
#   2026-07-22時点、Google Drive/Sheets書き込みはサービスアカウントではなくOAuth 2.0
#   ユーザー認証（リフレッシュトークン方式）に統一した（common/google_oauth_auth.py参照。
#   サービスアカウントは個人のGoogle Drive保存容量を持たず、新規ファイル作成が
#   403 insufficientParentPermissionsで失敗するため）。本番実行（GitHub Actions）で必要な
#   のはgoogle-authのgoogle.oauth2.credentials.Credentialsのみで、google-auth-oauthlibは
#   scripts/generate_google_oauth_token.py（リフレッシュトークンの初回発行、ローカル専用）
#   でのみ使用する。
# 全レイヤー共通: python-dotenv（ローカル開発時の.env読み込み）, jsonschema（各レイヤーのJSON契約検証）
# テスト: pytest
